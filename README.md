## Technical and ML Writing

### Poetic Form Classification Using a BERT + BiLSTM Model
**Document type:** ML research report

**Audience:** Technical, NLP/ML practitioners and researchers

**Role:** Sole author

Research report on a hybrid BERT-BiLSTM architecture for classifying poems into nine poetic forms. Covers dataset construction from multiple sources (2,270 poems), model architecture decisions, class imbalance strategies, and a detailed performance comparison across four model iterations.

## Technical Communication

### Proposal: SMB Platform Enhancements
**Document type:** Internal technical proposal

**Audience:** Internal stakeholders (Head/VP of Product, Product Manager)

**Role:** Sole author; also designed the document layout and created supporting visualizations in Tableau

**Context:** Writing assignment from Technical Communication II, UC Berkeley Extension

A 33-page proposal for a fictional SaaS-based payments platform (MerchBill), recommending the addition of a push notification microservice and dunning management features. Includes an executive summary, industry and market analysis, system architecture, implementation plan with timeline and financials, and a competitive landscape assessment.
